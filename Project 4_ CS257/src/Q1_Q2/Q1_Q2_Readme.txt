Overview:
This program implements and tests generic searching algorithms (LinearSearch and BinarySearch) in Java.

Instructions:
Assuming that JDK is installed on your machine, follow the steps below to run the program:
Compile the java files by running the command: javac Q1_Q2/*.java
Run the test program by running the command: java Q1_Q2.SearchTest <filename>
where <filename> is the name of the file containing the array of strings.

Results:
The program successfully searches for a word in given file using both the Linear Search and Binary search algorithms.

Known Bugs:
There are no known bugs at this time.
